Using the table

Each row contains a swatch for a particular color at various lightness levels. We compute the lightness on a scale of 0 (black) to 100 (white). Note this is perceived lightness, which takes into account irregularities in vision for different hues. This allows you to change one variable at a time (hue or lightness) when changing colors.

Click a color in the table to copy the hex color code to your clipboard. The table displays raw text as well so you can manually copy them (CTRL/CMD-C). Do this when you need an entire switch or the entire table.

Configuring the table

Toggle the show configuration switch if it is not already on. Then you can make adjustments using the sliders or by dragging the hue markers on the color wheel. Each marker represents the hue and saturation of a swatch. The edge of the wheel is fully saturated and the center of the wheel is fully unsaturated (white). The sliders have the following effects:

count:
The number of hues or swatches to include in the palette.
distribution:
Distributes hues evenly around the wheel or near a specific hue set by the offset. This control is disabled if count is set to 1.
offset:
A hue offset that applies to all swatches.
saturation:
Color saturation level that applies to all swatches.
range/step:
The upper and lower bounds of lightness and spacing between steps.
Special features

Try going to rgbpalette.com/red, rgbpalette.com/purple, or rgbpalette.com/#ffA500, etc. We'll take the color and build a swatch out of it!

Once you have created a palette, save a link to it with the link button.

Toggle the complementary switch at the top of the palette to show complementary colors. These are colors on the opposite side of the color wheel that match well.

Turn on the alt shades switch to show shades in terms of darkness ranging from 0 (white) to 1000 (black).
